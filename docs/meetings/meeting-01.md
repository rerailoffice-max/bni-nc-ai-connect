## チーム方針・基本理念

### 目的
1. **参加メンバーのビジネス発展** — 時間の浪費ではなく、投資になる活動
2. **仲間同士の協力と相互支援** — ライバルではなく、協力し合える仲間
3. **AI最新情報の共有** — 発見・新サービス・技術動向の情報交換
4. **コラボ・協業の創出** — プロジェクト型でチームを組んで成果を出す
5. **社会に役立つAIチーム** — 自己だけでなく、社会全体・日本・世界に貢献

### ビジョン
> **「令和のトキワ荘」** — ここから将来の有名な方々が発掘される、切磋琢磨するチーム

### 運営ルール
- **仲間の悪口・陰口は厳禁**（違反者は退場）
- 不満は「意見」として建設的に出す
- プロジェクトへの参加は自由意思（強制なし）
- BNIのブランド・ロゴの不正使用を避ける（リージョン内活動として進める）

### 注意事項（上川氏の経験談）
- リージョンの壁：他リージョンとの活動は指摘を受ける可能性がある
- BNIロゴの使用制限：公式ブランドの勝手な使用はNG
- 政治的配慮：将来的にナショナル本部の承認を得る方向で進める

## ナショナルカンファレンス前夜祭プロジェクト討議

### イベント概要

| 項目 | 内容 |
|------|------|
| **イベント名** | ナショナルカンファレンス前夜祭（インターナショナルナイト） |
| **日時** | 2026年4月27日（月）17:00〜21:00 |
| **会場** | 汐留 |
| **規模** | 最大600名参加予定 |
| **形式** | 立食パーティー形式のネットワーキングイベント |
| **設備** | 約50テーブル、約30ブース |
| **参加者** | 全国のBNIメンバー＋海外（台湾・香港・中国・マレーシア・インドネシア等）100名程度 |
| **企画担当** | 東京港中央リージョン（上川氏が実行委員） |
| **予算** | ナショナル本部から予算が出ない可能性あり |

### 認識された課題

1. **大人数での交流の困難さ** — 600名規模で知らない人ばかり、自分から行きにくい
2. **名刺交換後のフォローアップ不足** — もらった名刺がデータ化されず放置
3. **マッチング機会の欠如** — 会いたい人がどこにいるかわからない
4. **会場の動線問題** — 人が多すぎて動けない、ガチャガチャする
5. **参加後の満足度** — 「参加したけど全然つながらなかった」という声（前年の参加者から）

### ミーティングで出されたアイデア

1. **番号制マッチングシステム**（上川・勝又）
   - 参加者全員に番号を付与
   - AIが相性の良い組み合わせをランク付け
   - テーブル配置で100番台は固定、200番台が移動する方式

2. **事前オンライン顔合わせ**（上川）
   - イベント前にグループを作り、5〜10人でオンラインミーティング
   - 「温まった状態」で当日の対面を迎える

3. **名刺デジタル化＋インフォグラフィック**（酒井）
   - 名刺を撮影 → 画像認識でデータ化
   - 相手の事業情報をビジュアル化（インフォグラフィック/スライド形式）
   - ワンアクションで次のつながりに

4. **リファーラルAI連携**（大城のサービス活用可能性）
   - 参加者情報をAIで分析 → マッチングスコア算出
   - 紹介依頼の自動化

5. **3段階アプローチ**（上川：インバウンドマーケティングの「旅前・旅中・旅後」応用）
   - **事前（Before）**: 参加者情報収集、事前マッチング、オンライン顔合わせ
   - **当日（During）**: AIマッチング、テーブル配置最適化、名刺デジタル化
   - **事後（After）**: フォローアップ自動化、つながり深化サポート

### プロジェクト参加意思

**全員参加を表明**（温度差はあるが、できる範囲での貢献を約束）
