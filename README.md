# AI Connect — BNI前夜祭 コンタクトサークル自動編成システム

BNIナショナルカンファレンス前夜祭（2026/4/27）で使用する、AIによるコンタクトサークル（パワーチーム）自動編成システム。

## 概要

事前アンケート（7問）の回答を **Gemini 3.1 Flash** が分析し、ビジネスシナジーの高い **5-7名グループ** を自動編成する。

## システム構成

```
Googleフォーム（7問）→ スプレッドシート → GAS → Gemini Flash API → スプレッドシート（結果）
                                                                     → スクリーンHTML
                                                                     → 名前検索HTML
```

## ディレクトリ構造

```
docs/
  proposal/         — 提案書（AI Connectコンセプト + NC全体提案）
  requirements/     — 要件定義書
  meetings/         — 会議議事録・アジェンダ
src/
  gas/              — Google Apps Script コード
  screen/           — スクリーン表示HTML + 名前検索HTML
data/               — ダミーデータ・テストデータ
```
